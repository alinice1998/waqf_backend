# Waqf Backend - محرك التقسيم بالوقفات

هذا المستودع يحتوي على الواجهة الخلفية (Backend) لمشروع **وقف**، وهو نظام ذكي لتقسيم التلاوات القرآنية آلياً باستخدام الذكاء الاصطناعي بناءً على وقفات النفس.

يعتمد المحرك على نموذج `WhisperX` المتخصص في التعرف على الكلام، مع استخدام خوارزميات متقدمة لاكتشاف فترات الصمت (Silence Detection) مثل `Librosa` و `Silero VAD` لتحديد مواضع الوقف بدقة في التلاوة.

## الميزات الأساسية (Features)
- **المحاذاة الزمنية (Alignment):** محاذاة دقيقة للنص القرآني مع التلاوة الصوتية.
- **اكتشاف الوقفات (Silence Detection):** يعتمد على محركات متعددة لاستشعار وقفات القارئ (مثل Librosa و Silero).
- **التعرف على الكلام:** استخدام نموذج `WhisperX` لرفع دقة التوقيتات على مستوى الكلمة.
- **واجهة برمجية (API):** مبنية باستخدام `FastAPI` لمعالجة التلاوات بسرعة وكفاءة.

## التشغيل السحابي عبر Google Colab
تم تصميم هذا المحرك ليعمل بسهولة كخادم سحابي باستخدام **Google Colab** لتوفير موارد GPU اللازمة للذكاء الاصطناعي.
الواجهة الأمامية للمشروع تزودك بالأكواد اللازمة لتشغيل الخادم مباشرة.

## متطلبات التشغيل (Requirements)
- Python 3.10+
- PyTorch (مع دعم CUDA)
- FastAPI & Uvicorn
- WhisperX & Librosa & soundfile

## كيفية التشغيل محلياً (Local Setup)
في حال أردت تشغيل المحرك على جهازك المحلي الذي يحتوي على كرت شاشة (GPU):
1. قم بتثبيت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```
2. شغل خادم FastAPI:
   ```bash
   uvicorn server.app:app --host 0.0.0.0 --port 8000
   ```
